== README

* Search for through a listing of restaurants with the help of the searchkick gem. 

* Add a rating system and calculate the average rating of a restaurant with jQuery raty.

* Integration with Google Maps API to display Restaurant locations.

* Allow Users to write a review.

* Bootstrap for front-end design.

* Admin setting to add new restaurants.

* Image hosting with Amazon S3.

* Deployment with Heorku
